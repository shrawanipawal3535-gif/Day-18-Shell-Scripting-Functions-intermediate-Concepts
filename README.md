# Day-18-Shell-Scripting-Functions-intermediate-Concepts
Functions are used to organize and reuse code in a script. They make scripts cleaner, modular, and easier to maintain Functions can accept arguments (like $1, $2) to perform dynamic operations The local keyword is used inside functions to prevent variables from affecting the global scope.
Advanced scripts use strict mode to improve safety and prevent unexpected failures Functions are commonly used in DevOps scripts for system checks, automation tasks, logging, and monitoring.
